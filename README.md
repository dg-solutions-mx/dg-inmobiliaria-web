# DG Inmobiliaria – Home Astro

Landing page visual inspirada en el mockup coastal premium de DG Inmobiliaria Mazatlán.

## Ejecutar

```bash
npm install
npm run dev
```

## Compilar

```bash
npm run build
```

## Personalizar antes de publicar

- Sustituye `WHATSAPP_URL` en `src/components/WhatsAppButton.astro` y `src/components/Header.astro`.
- Sustituye correos, redes y aviso de privacidad en `src/components/Footer.astro`.
- Reemplaza imágenes secundarias dentro de `public/assets/` cuando tengas fotografías reales.
- El formulario es visual; aún no envía información a un backend.
